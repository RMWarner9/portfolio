Personal Wellness Tracker

About The Project:

The Personal Wellness Tracker is a small Python application that uses a local, single-file database for storing wellness information. Created for the people who have privacy in mind, all data is stored locally so there is no concern over sharing information with third-parties.

The application allows for the user to store financial, and emotional information for the user to get a visual of their financial and emotional wellness. It also allows the user to store journal entries.

My Garden App

About The Project:

My Garden App is an application that makes taking care of your plants-- indoor or outdoor-- easy!
The application's backend is written in Python and utilizes FastAPI to expose a series of endpoints for the user to get plant information. It uses PerenualAPI for searching plant information and stores it locally in a sqlite database.
The application has many helpful features such as care notifications, plant monitoring with sensor integration, and weather notifications.
