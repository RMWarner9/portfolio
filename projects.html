Personal Wellness Tracker

About The Project:

The Personal Wellness Tracker is a small Python application that uses a local, single-file database for storing wellness information. Created for the people who have privacy in mind, all data is stored locally so there is no concern over sharing information with third-parties.

The application allows for the user to store financial, and emotional information for the user to get a visual of their financial and emotional wellness. It also allows the user to store journal entries.
